# TPI-Biblioteca-PA2026-
El presente trabajo muestra en el desarrollo de un sistema de gestión de biblioteca orientada a objetos.  
El sistema permite administrar libros, estudiantes y prestamos realizados dentro de una biblioteca fisica. 

La solución fue diseñada aplicando conceptos fundamentales de POO, patrones de diseño y modelado de UML, cumpliendo con los requisitos cumplidos en la consigna. 

# Problematica a Resolver 
Las bibliotecas necesitan realizar un control de:
   - Libros disponibles 
   - Usuarios registrados
   - Prestamos activos
   - Devoluciones 

Realizar este control manualmente puede provocar errores y pérdida de información. 
El sistema desarrollado permite automatizar estas tareas mediante objetos que representan las entidades reales del dominio.

# Objetivos 

**Objetivo general** 

Desarrollar una aplicación de consola que permita gestionar recursos de una biblioteca. 

**Objetivo Específico**
- Registrar libros
- Registrar estudiantes
- Realizar préstamos
- Registar devoluciones 
- Aplicar conceptos de POO
- Utilizar un patrón de diseño
